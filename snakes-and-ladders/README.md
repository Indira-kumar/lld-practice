## Feature Requirements
- Game will be played amongst multiple human players. No bots.
- Every player uses a third-party dice and can return any number.
- Board can be of any size.
- There can be different types of entities:
    - Snake (takes you down)
    - Ladder (takes you up)
    - Frog (jumps randomly)
- The positions of entities/special characters are not given by the client, but are decided randomly by the app at the start of the game.
Condition: the combination of positions of different characters must be valid. Snake head, and ladder tail cant be at the same location.
- Every player will have multiple pawns/pieces on the board.
- When will game start for a player: when they roll 1.
- When will game end for a player: when they reach board end
